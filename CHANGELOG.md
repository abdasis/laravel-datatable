## [Unreleased]

## [1.0.0] - 2025-01-03

### Added
- Initial release
- Server-side pagination with Laravel
- Global search across multiple columns
- Column-specific filtering (text, select, date, date range)
- Multi-column sorting with priority
- Column visibility toggle
- Column reordering via drag & drop
- Column pinning (left/right)
- Table preferences persistence to localStorage
- Mobile responsive design with card view
- Row selection functionality
- Integration with Spatie Laravel Data
- Integration with TanStack Table v8
- Inertia.js support with partial reloads
- Comprehensive TypeScript types
- shadcn/ui based components

### Features
- DatatableService for backend processing
- React Datatable component with full customization
- Auto-discovery via Laravel Package Auto-Discovery
- Publishable configuration and assets
- Complete documentation and examples
